# Compressed Infinity

18段階の鉄圧縮と、圧倒的な力を手に入れるMinecraft Mod。

## 特徴
- **18段階の鉄圧縮**: 膨大な鉄を一つのブロックに。
- **究極の装備**: 鉄から生まれる神話級の武器と防具。
